Découvrez un projet universitaire qui m'a beaucoup appris! LoQuali est un projet que j'ai chéri lors de ma deuxième année de BUT informatique. Ma mission? Créer une plateforme Bretonne de location de logements qui ferait rougir AirBnB de jalousie!

Les "clients" (oui, entre guillemets parce qu'en réalité, c'étaient mes profs) m'ont donné leurs exigences. Ensuite, c'était carte blanche pour le design du site. Moi, étant une créatif dans l'âme, j'ai opté pour une expérience utilisateur simple mais efficace. J'ai donc concocté une maquette du site, adaptée aussi bien pour les ordinateurs que les téléphones. Malheureusement, ma maquette est tellement grande que je ne peux pas vous la montrer clairement. Mais pas de panique! Le site final a été construit à son image.

Évidemment, comme c'est un projet scolaire, j'ai dû remplir le site avec un jeu de données fictif. Donc, désolé de vous décevoir, mais vous ne pourrez pas vraiment louer votre logement de rêve à partir de là. Mais bon, c'est le jeu, parfois on doit se contenter de rêver.
