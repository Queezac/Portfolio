PUISSANCE 4

Le code en C que j'ai développé est une implémentation du jeu Puissance 4 jouable dans un Terminal, offrant une expérience amusante pour un ou deux joueurs. Il propose une interface utilisateur interactive, permettant aux joueurs de placer leurs pions stratégiquement sur une grille et de s'affronter pour atteindre la victoire.

Jeu du puissance 4 - Code source langage C

Le code ci-dessus, représente un jeu du Puissance 4 conçu pour deux joueurs. Les règles sont simples : chaque joueur doit aligner quatre pions de son signe ('X' ou 'O') horizontalement, verticalement ou en diagonale pour remporter la partie.
Pour se déplacer sur la grille de jeu, les joueurs utilisent les touches 'q' pour se déplacer vers la gauche et 'd' pour se déplacer vers la droite. Pour placer un pion, ils utilisent la touche 'espace'.
Une particularité de ce code est qu'en cas de tentative de déplacement à gauche lorsqu'on se trouve déjà tout à gauche, le joueur réapparaît tout à droite de la grille, créant une boucle de déplacement fluide et sans interruption, et vice versa pour la droite.
Le jeu présente deux résultats possibles : la victoire d'un joueur en alignant quatre pions de son signe, ou une partie nulle lorsque la grille est entièrement remplie sans qu'aucun joueur n'ait réussi à aligner quatre pions.
De plus, si un caractère autre que 'q', 'd' ou 'espace' est saisi lors du déplacement, un message d'erreur est affiché pour indiquer une entrée invalide, permettant ainsi aux joueurs de corriger rapidement leur saisie.

Une des fonctionnalités en plus, intéressantes du code qui n'est pas présente ci dessus, est la possibilité de choisir entre 1 ou 2 joueurs. En modifiant le code, il est donc possible d'ajouter une option permettant aux joueurs de jouer contre l'ordinateur.
Dans ce mode de jeu contre l'ordinateur, une fonction stratégique à été créée pour l'IA. Cette fonction permet à l'ordinateur de bloquer les mouvements du joueur avant qu'il n'aligne quatre de ses pions, tout en essayant également de calculer les mouvements optimaux pour aligner ses propres pions.

En intégrant ces deux morceaux de code, la possibilité de jouer au Puissance 4 contre l'ordinateur devient réalisable.
D'une part, le code de gauche implémente la stratégie que l'ordinateur utilisera contre le joueur. Il permet à l'ordinateur de prendre des décisions tactiques pour bloquer les mouvements du joueur et maximiser ses propres chances de victoire. D'autre part, le code de droite gère le déroulement du jeu, en alternant les tours entre le joueur et l'ordinateur.
